## ✨ Features
- Search any city worldwide
- Current temperature, feels like, humidity, wind & pressure
- 5-day weather forecast
- Humidity & wind speed charts
- °C / °F temperature toggle
- Recent search history
- Dynamic background that changes with weather condition
- Fully responsive on mobile and desktop

## 🚀 How to Run Locally
1. Clone the repo
   git clone https://github.com/rushikethana-28/skywatch-weather.git

2. Get a free API key from https://openweathermap.org

3. Open weather-dashboard.html in VS Code

4. Replace YOUR_API_KEY with your actual key

5. Run with Live Server extension
